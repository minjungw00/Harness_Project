# 접점별 사용법

이 문서는 현재 문서 세트의 접점별 사용법을 담당합니다. 사용 문서일 뿐이며 커넥터 계약, API 스키마, 저장소 계약, 보안 증명, 구현 안내가 아닙니다.

## 담당하는 것 / 담당하지 않는 것

이 문서가 담당합니다.

- 이름 붙은 접점을 통해 작업할 때의 사용자용, 에이전트용 사용법
- 접점을 사용할 수 없거나, 역량이 부족하거나, 역량이 제한된 상황의 접점별 표현
- 접점이 하네스에 무엇을 확인해 달라고 할 수 있는지에 대한 사용법 수준 안내
- 실제 사용 상황에서 정확한 참조 담당 문서로 가는 링크

이 문서는 담당하지 않습니다.

- 커넥터 동작, `capability_profile`, 맥락 패킷 규칙, 대체 동작: [에이전트 통합](../reference/agent-integration.md)
- 공개 API 메서드나 스키마: [참조 색인](../reference/README.md)
- 보안 보장: [보안](../reference/security.md)
- 저장소, 아티팩트, 상태 보기: 관련 참조 담당 문서

## 사용법 경계

사용법은 실무적이어야 하지만 규범 계약이 되면 안 됩니다. 사용법에 정확한 메서드 동작, 스키마 필드, 저장 효과, 보안 보장, 템플릿 본문이 필요하면 먼저 담당 문서에 계약을 추가한 뒤 이 문서에서 링크합니다.
