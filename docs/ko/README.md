# 하네스 문서

이 문서는 하네스 한국어 문서의 진입점입니다. 독자가 알맞은 문서로 이동하도록 안내하며, 기술 계약을 직접 정의하지 않습니다.

## 여기서 시작하기

처음 읽는 독자는 [시작하기](start.md)를 먼저 봅니다. 하네스가 어떤 일을 돕는지와 문서 묶음의 흐름을 잡을 수 있습니다.

## 역할별 읽기 경로

- 실제 작업을 진행하는 사용자는 [사용자 가이드](use/user-guide.md)를 봅니다.
- 에이전트는 [에이전트 가이드](use/agent-guide.md)를 봅니다.
- 판단이 필요한 상황의 예시는 [판단 예시](use/judgment-examples.md)에서 확인합니다.

## 현재 범위

현재 MVP의 경계는 [현재 MVP 범위](reference/active-mvp-scope.md)가 담당합니다. 전달 범위 판단은 이 README가 아니라 담당 문서에서 확인합니다.

## 참조 계약

기술 계약을 확인해야 하는 독자는 [참조 색인](reference/README.md)에서 시작합니다. 참조 색인은 각 계약 주제의 담당 문서로 안내하고, 이 진입점은 짧은 경로 안내로만 남습니다.

## 유지보수와 번역

유지보수자는 [작성 가이드](maintain/authoring-guide.md), [문서 점검](maintain/checks.md), [번역 가이드](maintain/translation-guide.md)를 사용합니다. 영어와 한국어 문서는 줄 단위 번역이 아니라 의미 일치를 기준으로 함께 관리합니다.
