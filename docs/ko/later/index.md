# 이후 후보 색인

이 페이지는 이후 후보 목록만 담당합니다. 현재 MVP 동작, 구현 작업, 런타임 권한, 현재 API 계약, 현재 저장 효과, 현재 보안 보장을 정의하지 않습니다.

현재 범위는 [현재 MVP 범위](../reference/active-mvp-scope.md)를 봅니다.

이곳의 언급은 승격이 아닙니다. 후보는 현재 범위와 관련 담당 문서가 같은 문서 전용 작업 묶음에서 갱신된 뒤에만 현재 MVP로 들어갈 수 있습니다.

프로필 조건부 값과 이후 후보를 구분합니다. 어떤 값이 현재 프로필에서 조건부로 활성화되려면 현재 범위와 담당 문서가 그 프로필과 지원 값 집합을 함께 이름 붙여야 합니다. 이 페이지의 값은 그런 승격 전까지 이후 후보입니다.
