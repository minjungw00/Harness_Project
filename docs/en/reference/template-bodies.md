# Template Bodies

This document owns exact rendered template bodies for active current MVP displays. It is documentation source material only and does not generate projection files, runtime artifacts, QA records, acceptance records, or close records.

## Owns / Does not own

This document owns:

- exact active template bodies for rendered status or support displays
- user-facing template wording boundaries
- locale-aware rendered labels where a template needs them
- links from template placeholders to schema owners

This document does not own:

- projection authority, freshness, or read-only derived-display rules; see [Projection And Templates](projection-and-templates.md)
- source-of-truth state or storage; see [Core Model](core-model.md) and storage owners
- API schemas or value sets; see API schema owners
- later template candidates; see [Later Candidate Index](../later/index.md)

## Boundary

Template text is display text. Editing rendered Markdown, status cards, managed blocks, or template output does not mutate Core state and does not create Write Authorization, evidence, final acceptance, residual-risk acceptance, or close readiness.
