# Harness Project / 하네스 프로젝트

Harness is an agency-preserving local operating kernel for AI-assisted development. This repository currently holds the reader-centered documentation set prepared for human review.

Harness는 AI 지원 개발에서 사용자의 판단권을 보존하는 로컬 운영 커널입니다. 이 저장소는 사람의 검토를 위해 준비된 독자 중심 문서 세트를 담고 있습니다.

## Current Phase / 현재 단계

- The initial documentation baseline has been reviewed.
- The documentation set has been redesigned around reader paths and practical entrypoints.
- The redesigned documentation is ready for human review.
- Harness server and runtime implementation have not started.
- Implementation must start only after the redesigned documentation is accepted.

- 초기 문서 baseline 검토가 완료되었습니다.
- 문서 세트는 독자 경로와 실용적인 진입점을 중심으로 재설계되었습니다.
- 재설계된 문서는 사람의 검토를 받을 준비가 되어 있습니다.
- Harness server와 runtime 구현은 아직 시작하지 않았습니다.
- 구현은 재설계된 문서가 승인된 뒤에만 시작해야 합니다.
