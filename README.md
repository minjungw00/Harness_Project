# Harness Project / 하네스 프로젝트

Harness is an agency-preserving local operating kernel for AI-assisted development. This repository currently holds the completed initial documentation baseline for the pre-MVP Harness design.

Harness는 AI 지원 개발에서 사용자의 판단권을 보존하는 로컬 운영 커널입니다. 이 저장소는 현재 pre-MVP Harness 설계의 초기 문서 baseline 작성을 완료한 상태입니다.

## Current Phase / 현재 단계

- Initial documentation authoring is complete.
- The next step is human review of the documentation set.
- Harness server and runtime implementation have not started yet.
- After human review, the Harness server should be built from the reviewed documentation.

- 초기 문서 작성은 완료되었습니다.
- 다음 단계는 문서 세트에 대한 인간 검토입니다.
- Harness server와 runtime 구현은 아직 시작하지 않았습니다.
- 인간 검토 이후, 검토된 문서를 기준으로 Harness server를 구축합니다.
